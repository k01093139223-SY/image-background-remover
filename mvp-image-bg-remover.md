# 图片背景去除工具 - MVP 需求文档

**版本：** v1.0  
**日期：** 2026-03-26  
**状态：** 初稿

---

## 1. 产品概述

| 项目 | 内容 |
|------|------|
| **产品名称** | Image BG Remover |
| **产品定位** | 轻量级在线图片背景去除工具 |
| **核心功能** | 用户上传图片 → AI 自动去除背景 → 预览/下载结果 |
| **目标用户** | 设计师、电商卖家、内容创作者 |

### 核心约束
- **无存储**：图片纯内存处理，不持久化
- **部署平台**：Cloudflare (Workers + Pages)
- **AI 能力**：Remove.bg API

---

## 2. 功能需求

### 2.1 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 图片上传 | 支持拖拽 + 点击上传，格式：PNG/JPG/WEBP | P0 |
| 背景去除 | 调用 Remove.bg API 处理图片 | P0 |
| 结果预览 | 展示原图 vs 去除背景后的对比 | P0 |
| 图片下载 | 一键下载去除背景后的 PNG 图片 | P0 |
| 加载状态 | 处理中显示进度/骨架屏 | P1 |
| 错误处理 | API 失败、网络异常的用户提示 | P1 |

### 2.2 非功能需求

| 需求 | 描述 |
|------|------|
| **响应时间** | < 10 秒（取决于 API） |
| **图片大小** | 最大 10MB |
| **并发限制** | 免费额度内（Workers 1000req/天） |
| **移动端适配** | 支持手机浏览器使用 |

## 3. 技术架构

### 3.1 系统架构

```
┌─────────────┐      ┌──────────────────┐      ┌─────────────┐
│  用户浏览器  │ ──▶  │  Cloudflare      │ ──▶  │ Remove.bg  │
│  (前端页面)  │      │  Workers (后端)  │      │   API       │
└─────────────┘      └──────────────────┘      └─────────────┘
       ◀──────────────── ────────────────────── │
              返回处理后的图片(PNG)              │
```

### 3.2 技术栈

| 层级 | 技术选型 |
|------|----------|
| **前端** | Cloudflare Pages + 原生 HTML/CSS/JS |
| **后端** | Cloudflare Workers |
| **API** | Remove.bg API |
| **部署** | Cloudflare (免费额度) |

### 3.3 成本预估

| 项目 | 免费额度 | 超出费用 |
|------|----------|----------|
| Cloudflare Workers | 1000 req/天 | $0.000000015/请求 |
| Cloudflare Pages | 500MB 带宽 | $0.01/GB |
| Remove.bg API | 无 | $0.09/张 |

## 4. 界面设计

### 4.1 页面结构

```
┌────────────────────────────────────┐
│           顶部导航                  │
├────────────────────────────────────┤
│                                    │
│         [上传区域]                  │
│      点击或拖拽上传图片              │
│                                    │
├────────────────────────────────────┤
│    [原图]          [处理结果]        │
│    ◉                            ◉   │
├────────────────────────────────────┤
│         [下载结果] 按钮             │
└────────────────────────────────────┘
```

### 4.2 交互流程

1. 用户进入页面
2. 上传/拖拽图片
3. 点击"去除背景"
4. 等待处理（显示 Loading）
5. 展示对比结果
6. 点击下载保存
